# RETRY 🕹

> 중고 게임 거래 플랫폼

RETRY(리트라이)는 사용자가 플레이한 게임을 다른 사용자에게 판매할 수 있는 중고 게임만을 위한 특화 플랫폼입니다. 이 플랫폼은 자신에게 맞지 않거나, 이미 플레이를 한 게임을 다른 사용자에게 판매하여 다시한번 플레이 팔수 있는 기회를 마련합니다.

- 🎮 중고 게임 거래 및 판매
- 📈 게임 데이터베이스 구축 및 시세파악
